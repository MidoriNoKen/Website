Instalasi Linux Server Menggunakan AWS Elastic Computing

Seperti yang kita tau sebelumnya, server secara konvensional haruslah membutuhkan sebuah ruangan khusus secara fisik disuatu tempat. Jika kita bertanya tentang server secara konvensional, pasti dalam pembuatannya membutuhkan biaya yang sangat besar, seperti biaya bangunan, pemasangan hingga perawatan di tiap harinya. Belum juga terlintas dipikiran kita setelah server telah dibentuk, yakni mengenai berguna atau tidaknya server yang kita buat dan memori hingga processor yang tidak terpakai secara efektif dan efisien. Dan masih banyak lagi. Amazon Web Service inilah yang dapat membantuk kita dalam menangani masalah biaya yang besar ini.
